Space - Пространства для приложений
  = Core\Space\
    - class Manager

      -- Раздел ключ-значение --

      ::GetKeys( 'vendor.app' ) - Возвращает все ключи данного пространства
      ::GetKey( 'vendor.app.key' ) - Возвращает значение ключа
      ::GetValues( 'vendor.app.key' ) - Возвращает все варианты значений для данного ключа
      ::SelectValue( 'vendor.app.key' , 'value' ) - Присваивает ключу выбранное значение из доступных

      -- Раздел коллекции --

      ::GetCollections( 'vendor.app' ) - Возвращает все коллекции данного пространства
      ::GetCollection( 'vendor.app.collection' ) - Возвращает коллекцию
      ::CollectionItemStatus( 'vendor.app.collection.item' , bool ) - Возвращает коллекцию

    - class Build

      ::force();

      Во время билда сохраняется источник значения,
     тобиш Путь к SpaceBundle.php из которого значение было добавлено
        Значит мы можем
          - При удалении приложения изымать только его значения
          - При обновлении SpaceBundle приложения,
            мы можем обновлять его только для одного приложения
          - При установке приложения значения будут осторожно
            добавленны в пространства
      Для коллекций создаётся поле enabled,
     благодаря этому мы можем активировать,
     либо дизактивировать любой объект из коллекции
